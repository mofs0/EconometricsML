# 2. 工具变量（IV）/两阶段最小二乘（2SLS）

理论概述：当解释变量与误差项相关时，OLS 有偏。使用工具变量 $Z$ 满足相关性与外生性，通过两阶段回归得到一致估计。

数学公式：第一阶段 $X=Z\pi+v$；第二阶段 $y=X\beta+\varepsilon$，2SLS 解为
$$\hat{\beta}=(X^TP_ZX)^{-1}X^TP_Zy,\quad P_Z=Z(Z^TZ)^{-1}Z^T.$$ 

Python 简要示例：

```python
from linearmodels.iv import IV2SLS
iv = IV2SLS(dependent=y, exog=exog, endog=endog, instruments=Z).fit()
print(iv.summary)
```

中文论文示例：王五，《工具变量在宏观冲击识别中的应用》，2018。
