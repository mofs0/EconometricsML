# 计量经济学示例

这里放偏传统计量经济学的 Python 示例与 Jupyter notebooks。

## 当前内容

### Python 示例

- `01_ols_regression.py` - OLS 回归与统计检验

### Jupyter notebooks

- `01_OLS_Angrist2008_clean.ipynb` - OLS 详尽示例
- `02_IV.ipynb` - 工具变量 / 2SLS
- `03_FixedEffects.ipynb` - 固定效应模型
- `04_RandomEffects.ipynb` - 随机效应模型
- `05_DiD.ipynb` - 差分中的差分
- `06_GMM.ipynb` - 广义矩估计
- `07_ARIMA.ipynb` - ARIMA 时间序列模型
- `08_VAR.ipynb` - VAR 向量自回归
- `09_GARCH.ipynb` - GARCH 波动率模型
- `10_ProbitLogit.ipynb` - 二项选择模型

## 推荐阅读顺序

1. 先看 `01_OLS_Angrist2008_clean.ipynb`，它是所有计量 notebook 的模板。
2. 再按编号阅读 `02_IV.ipynb` 到 `10_ProbitLogit.ipynb`。
3. 如需了解 notebook 结构要求，请看 `NOTEBOOK_TEMPLATE_GUIDE.md`。
