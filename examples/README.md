# Examples 总览

本目录按主题组织示例和模型说明（适合教学与快速查阅）：

- 计量经济学（examples/econometrics/）
- 机器学习（examples/ml/）
- 深度学习（examples/deep_learning/）
- 强化学习（examples/rl/）

每个模型文件包含：理论概述、关键数学公式、简要 Python 示例、以及使用该模型的中文论文示例。
