# Python 教程索引

这个目录用于放 Python Notebook 教程，按“从简单到复杂”组织。

## Notebook 目录

1. [01_data_preparation.ipynb](01_data_preparation.ipynb) - 数据读取、清洗和导出
2. [02_ols_basics.ipynb](02_ols_basics.ipynb) - OLS 与基础计量回归
3. [03_time_series.ipynb](03_time_series.ipynb) - VAR 与 GARCH
4. [04_ml_prediction.ipynb](04_ml_prediction.ipynb) - 机器学习预测与评估
5. [05_econometrics_family.ipynb](05_econometrics_family.ipynb) - 计量模型进阶总览
6. [06_ml_family.ipynb](06_ml_family.ipynb) - 机器学习模型进阶总览
7. [07_deep_learning_family.ipynb](07_deep_learning_family.ipynb) - 深度学习模型总览
8. [08_rl_family.ipynb](08_rl_family.ipynb) - 强化学习模型总览

## 建议的教程结构

1. 数据读取与清洗
2. OLS 与基础计量回归
3. 时间序列与波动率模型
4. 机器学习与预测评估
5. 计量 / 机器学习 / 深度学习 / 强化学习 四类模型总览

## 和 Stata 的分工

- Python 教程负责：数据处理、建模、可视化、结果整理
- Stata 工作流负责：回归、面板分析、论文表格输出

如果你主要做实证研究，可以先看 Python 教程，再切到 [Stata 联动工作流](../docs/stata-workflow.md)。
