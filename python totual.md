# python_tutorial/ 文件夹立项说明

## 定位

专为**经济 / 金融 / 商科**背景的零基础同学设计，内容紧贴本仓库（EmpirLab）的实际使用场景。
不追求大而全，只教"用得到的"。学完 P00–P05 即可独立运行本项目全部 Notebook。

---

## 文件清单与各课核心内容

### P00 写在前面：如何使用本教程
- Jupyter Notebook 界面介绍（Cell 是什么、怎么运行、快捷键）
- 如何看报错信息（读懂 Traceback 的最后一行）
- 遇到问题去哪里找答案（Google、Stack Overflow、ChatGPT）
- 本教程学习路线图

### P01 Python 基础：变量与数据类型
- 变量赋值（`x = 5`、`name = "张三"`）
- 基本数据类型：整数、浮点数、字符串、布尔值
- 列表（list）：增删改查
- 字典（dict）：键值对，类比 Excel 的"列名 → 数据"
- **经济学场景练习**：用字典存储一家企业的财务指标

### P02 Python 基础：条件与循环
- `if / elif / else` 条件判断
- `for` 循环遍历列表
- `while` 循环（简单介绍）
- 列表推导式（一行写出循环）
- **经济学场景练习**：批量计算多家企业的资产负债率并筛选高风险企业

### P03 Python 基础：函数与模块
- 定义函数（`def`）、参数、返回值
- 常用内置函数（`len`、`sum`、`range`、`print`、`type`）
- `import` 导入模块（以 `math`、`random` 为例）
- 如何读懂别人写的函数（看 docstring）
- **重点**：如何调用本项目的 `empirlab` 包（`from empirlab.traditional.ols import OLS`）

### P04 数据处理：NumPy 数组
- 创建数组（`np.array`、`np.zeros`、`np.ones`、`np.arange`）
- 数组运算（加减乘除、广播机制）
- 切片与索引
- 常用统计函数（`mean`、`std`、`max`、`min`）
- **经济学场景练习**：计算股票收益率序列的均值与标准差

### P05 数据处理：Pandas 表格操作（最重要！）
- 读取数据：`pd.read_csv`、`pd.read_excel`
- 查看数据：`df.head()`、`df.info()`、`df.describe()`
- 选取列与行：`df["列名"]`、`df.loc[]`、`df.iloc[]`
- 筛选数据：`df[df["列名"] > 某值]`
- 新增列、删除列
- 处理缺失值：`dropna`、`fillna`
- 分组统计：`groupby`
- 保存数据：`to_csv`、`to_excel`
- **经济学场景练习**：加载一份 A 股财务数据，筛选制造业企业，计算各年度平均 ROE

### P06 数据可视化：Matplotlib 画图
- 折线图（时间序列）
- 散点图（两变量关系）
- 直方图（分布）
- 箱线图（分组比较）
- 图表美化（标题、坐标轴标签、图例、中文显示）
- **经济学场景练习**：画出 GDP 增速时间序列图 + 工资与教育年限散点图

### P07 统计基础：描述性统计与假设检验
- 描述性统计（均值、中位数、方差、偏度、峰度）
- 正态分布直觉理解
- t 检验（单样本、双样本）
- 相关系数（Pearson）
- p 值怎么看？显著性水平怎么选？
- **与本项目对接**：这些概念在 OLS 回归结果表格里如何体现

### P08 实战演练：用 Python 复现一篇实证论文
- 选取一篇经典的 Mincer 工资方程论文
- 从数据读取、变量构造、OLS 回归到结果解读全流程
- 完整调用 `empirlab.traditional.ols` 模块
- 如何把代码结果写进论文（图表导出、系数汇报格式）
- **课后任务**：用自己的数据跑一遍，把结果贴到群里讨论

## 开发规范

- 每个 Notebook 开头有"本节你能学到什么"的学习目标
- 每节控制在 **30 分钟内**读完并可运行
- 全程使用经济/金融/商科的实际案例，不用"hello world"那种脱离场景的例子
- 每节末尾有 2–3 道练习题（有参考答案）
- 代码注释用中文，变量名用英文
- 不讲面向对象（OOP）、不讲装饰器等进阶内容，够用就好
